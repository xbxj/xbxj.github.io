反思与总结

Hexo博客
博客主题及选取原因：
美观（本博客使用fluid主题）
比起自己搭建博客，可以省去购买域名和服务器的成本，不需要考虑定期维护，对于不使用全栈的人（比如我），非常省力
博客页面布局及其设计思路
Home（个人主页）
Archives（博客中的主要文章和贡献）
Categories（将文章分类）
Tags（设置文章标签）
About（个人信息）
博客功能实现及其技术选择
热点信息记录（上传时间，热度，点赞数，评论）
附加功能（昼夜模式，多路径链接）
博客样式设计及其美学考量
图片（主页背景，个人信息页头像，网络角标）
博客制作过程中遇到的问题及其解决方法
问题1：hexo d始终没有办法将本地修改的内容与github上同步
解决方法：修改config_yml文件，将token一行注释掉。这个问题之前一直没法解决，之前每一次我都在github上反复生成ssh-key，但是无论选择key的可使用时间是多少，都会很快失效，非常麻烦。所以直接标注掉token这一行，反而解决了问题。
问题2：Failed to connect to github.com port 443:connection timed out
解决方法：
取消全局代理：
git config –global –unset http.proxy

git config –global –unset https.proxy
